# COFF

Offline Judge for Competitive Coding 

---

# How to run this Project?

1) Clobe the Repository.
2) Switch to `master` branch.
3) To build the project, enter `make`.
4) Run `./coff` .
5) To remove the program compiled in step 3, enter `make clean` .
